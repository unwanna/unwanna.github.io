Google Stock Price Predictor¶

This project uses machine learning, specifically LSTM (Long short-term memory) neural networks to predict the price of Google's closing price at the end of a trading day. The model is trained on historical data of Google's closing price. First a benchmark linear regression line is used to see how well the data fits within a linear dimmentsion. Since stocks do not move in a linear fashion a LSTM model is used which effectively creates a moving average for the data used to train the model.

import math
import pandas_datareader as web
import numpy as np
import pandas as pd
import datetime

def get_historical_data(symbol,start_date,end_date):
    ''' Daily quotes from Google. Date format='yyyy-mm-dd' '''
    symbol = symbol.upper()
    col_names = ['Date','Open','High','Low','Close','Volume']
    
    df = web.DataReader(symbol,data_source='yahoo',start=start_date ,end=end_date)
    return df
